DevTools MCP 简介

DevTools MCP（Model Context Protocol）是一个允许 DevTools 与外部运行时或服务协作的协议。使用 MCP Server 可以将 DevTools 扩展为展示工具、远程调试或自动化演示的平台。

Workshop 要点

  • 设置 MCP Server 并连接 DevTools
  • 在页面上触发自动样式或操作以便演示
  • 示例：通过 MCP 自动高亮页面元素 / 注入样式 / 触发点击
